# SQLSaturday #187 - Richmond 2013
Event Date: 09-03-2013 - Session time: 00:00:00 - Track: BI2
## Speaker: Melissa Coates
## Title: So You Want To Be A Rockstar Report Developer?
## Abstract:
### You don't want to settle for being an average Report Developer, right?  In this highly interactive session we'll discuss various development, standardization, deployment, and documentation practices that will make your SSRS development life easier, your output of higher quality, increase maintainability, and ultimately save you time.  Audience participation and sharing of experiences is encouraged as we cover choices you have for datasets, sorting, grouping, filtering, calculations, parameters, deployment, and delivery.  This session focuses primarily on SQL Server Reporting Services 2012, although some concepts may apply to other BI tools as well.
